CAPM Asset Management Dashboard

Risque, rendement, simulation de portefeuille et interprétation financière

Présentation du projet

Ce projet a été réalisé dans le cadre d’un travail de programmation appliqué à la finance. Son objectif est d’étudier la relation entre le risque et le rendement des actions à l’aide du modèle CAPM, tout en proposant une interface visuelle simple et élégante permettant de présenter les résultats de manière claire.

Nous avons utilisé Python pour charger les données financières, nettoyer les séries temporelles, calculer les rendements journaliers, estimer les bêtas des entreprises, déterminer les rendements théoriques prévus par le CAPM et comparer ces rendements aux résultats réellement observés. L’ensemble de ce travail est ensuite résumé dans ce dashboard.

Ce que contient le dashboard

La page Simulation permet à l’utilisateur de choisir un capital initial, une stratégie d’investissement et plusieurs entreprises. La page Résultats présente les indicateurs les plus importants du projet, notamment la fortune finale estimée, la probabilité de dépasser un objectif de gain et la comparaison entre plusieurs stratégies. La page Graphiques regroupe enfin les visualisations essentielles du modèle CAPM.

Entreprises étudiées

Les entreprises retenues dans le projet sont Apple, Microsoft, Tesla, Airbus, LVMH, TotalEnergies, Air Products, Alibaba, Meta Platforms et Philips. Nous avons volontairement choisi des entreprises issues de secteurs différents afin de comparer plusieurs profils de risque et de rendement. Le marché est représenté par SPY, utilisé comme approximation de l’indice S&P 500.

Intérêt financier du projet

L’intérêt principal du CAPM est de montrer qu’un actif financier ne doit pas être évalué uniquement à partir de son rendement, mais aussi à partir de son risque. Le bêta mesure cette sensibilité au marché, et le modèle permet ensuite de comparer le rendement observé d’une action au rendement théorique qu’elle devrait offrir selon son niveau de risque.

Cette approche permet de déterminer si une action semble sous-évaluée, correctement valorisée ou potentiellement surévaluée.
